# E-Commerce

A luxury-style static e-commerce storefront built with plain HTML, CSS, and JavaScript. The project is designed as a zero-build front-end experience with multiple sections, product browsing, cart interactions, and a polished editorial layout.

## Overview

This project includes:

- A premium landing page with hero, featured products, editorial content, testimonials, and newsletter sections
- Multi-page style navigation handled inside a single `index.html` file
- Product listing, filtering, sorting, and category browsing
- Product detail views with gallery, size, color, and quantity selection
- Cart and checkout-style flows powered by vanilla JavaScript
- Responsive layout for desktop and mobile screens

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Google Fonts

## Run Locally

Because this is a static site, there is no build step.

1. Clone the repository:

```bash
git clone https://github.com/vicky-69cyber/E-Commerce.git
```

2. Open the project folder.
3. Launch `index.html` in your browser.

You can also use a lightweight local server if you prefer, for example:

```bash
python -m http.server 5500
```

Then open `http://localhost:5500`.

## Deployment

This repository can be deployed directly on any static hosting platform, such as:

- Vercel
- Netlify
- GitHub Pages

Since everything is contained in `index.html`, no install or build command is required.

## Customization

You can easily adapt this project by editing:

- Product data and storefront behavior inside `index.html`
- Colors, typography, spacing, and layout styles in the embedded CSS
- Page sections such as the hero, featured products, lookbook, about, cart, and checkout views
